  [Node정보]
    - nodeName        : 노드 고유의 이름 반환(읽기전용 속성/get만 가능)
    - tagNeme         : 요소 이름 반환
    - nodeValue, data : 노드의 값(요소 노드 내부의 text정보를 얻기 위해서는 textNode까지 접근되어야 함)
    - textContent     : 노드의 값 반환(ElementNode까지만 접근해도 가능)
    - nodeType        : 노드의 타입, 유형(읽기전용 속성/get만 가능)
